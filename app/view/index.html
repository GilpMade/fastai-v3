This is a mushroom classifier for 29 different species of mushroom found in the Adelaide Hills. The server can only handle images up to 800kb in size so resize images to fit this if they are too large. This classifier is in its early stages and can still make mistakes so make sure to google the result and compare yourself!

This classifier does not attempt to provide advice on edibility of mushrooms and no mushroom that you are unsure of should ever be consumed.
If your image has been misclassified please email the image (as a jpeg) to gilpmade@gmail.com with the incorrect label and the correct classification of the mushroom. Please only send images of mushrooms found in the Adelaide Hills region.
